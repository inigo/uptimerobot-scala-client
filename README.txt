Use the UptimeRobot API to manage monitoring for our live servers.

== Building and testing ==

This is a Scala Activator project. Build and test with:

    bin/activator
    > compile
    > test

Before running tests, obtain an API key from http://uptimerobot.com/
and put that API key in the src/test/resources/test.conf file.
